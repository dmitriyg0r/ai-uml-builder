# Release Notes v0.1.3

Мы рады объявить о выпуске версии v0.1.3 приложения AI UML Builder!

## Исправления ошибок:

*   **Исправлена конфигурация CI/CD:** Добавлен недостающий npm скрипт `tauri`, который необходим для корректной работы автоматической сборки GitHub Actions. Это обеспечит успешную публикацию релизов в будущем.

Предыдущие изменения (из v0.1.2):
*   **Автоматическое обновление приложения:** Поддержка автообновления без переустановки.
*   **Улучшенная локализация:** Перевод интерфейса обновлений.
*   **Стабильность:** Улучшена загрузка чатов и исправлена инициализация плагинов.

Мы рекомендуем всем пользователям обновиться до последней версии!
